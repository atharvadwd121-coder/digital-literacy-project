## 📌 Project Overview
This project is based on **Digital Literacy** and aims to develop awareness about the safe and effective use of digital tools. It includes tasks related to infographic creation, digital portfolio building, coding platforms, professional email communication, and cybercrime awareness.

---

## 🧩 Task Summary

### Task 1 – Infographic
Created an infographic using **Canva** explaining digital literacy, safe internet practices, and useful tools for modern students.

### Task 2 – Digital Portfolio
Created professional profiles on **GitHub, LinkedIn, and Kaggle** to establish a credible digital presence and showcase AI research.

### Task 3 – Platforms
Solved beginner-level algorithmic problems on **HackerRank** and designed a comprehensive **Google Form quiz** to test digital literacy.

### Task 4 – Email Etiquette
Drafted professional emails for academic and career opportunities and created a **Social Media Do’s and Don’ts** checklist for students.

### Task 5 – Cybercrime Awareness
Developed a detailed case study on **UPI/Phishing Fraud** and authored a step-by-step **Prevention Checklist** for financial safety.
